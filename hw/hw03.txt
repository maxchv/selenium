В Selenium IDE составить тест:

1. Открыть сайт https://itstep.dp.ua/
2. Проверить наличие ссылки "Профессиональные курсы"
3. Перейти по ссылке "Профессиональные курсы"
4. Проверить наличие ссылки "Тестирование ПО"
5. Перейти по ссылке "Тестирование ПО"
6. Проверить наличие значения "Фамилия, имя" атрибута placeholder для первого поля ввода 
7. Проверить наличие значения "E-mail" атрибута placeholder для второго поля ввода 
8. Проверить наличие значения "Телефон" атрибута placeholder для третьего поля ввода 
9. Проверить наличие значения "Ваше сообщение..." атрибута placeholder для четвертого поля ввода 

При написании теста использовать XPath локаторы
