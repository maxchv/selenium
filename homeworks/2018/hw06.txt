Задание:

Создать базовый абстрактный класс Droid. Добавить абстракный метод info.
Добавить защищенное поле name строкового типа.
Добавить защищенное статическое поле count типа int.
Добваить поле id типа int.
Конструктор класса должен принимать имя робота.
В конструкторе реализвовать инкеремент count и присвоение id значения count.

Создать класс R2D2 - наследник Droid. В конструктор базового класса передать
имя робота. Реализовать метод info - для вывода информации о роботе: его имя и id.

Создать класс C3PO - наследник Droid. В конструктор базового класса передать
имя робота. Реализовать метод info - для вывода информации о роботе: его имя и id.

Создать массив роботов Droid. В цикле перебрать всех роботов и вызвать для каждого метод info()
